# Overwatch
[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/) 
![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-blue)

**Overwatch** is a chrome extension for YouTube that passively displays a timestamp of when the current video will finish in the middle of the video player, so its impossible to miss. 











## Features

- Always visible where your content is. 
- Automatically adjusts contrast for visibility on both light and dark backgrounds. 
- Customizable clock format. 

## Installation and Usage

    1. Download and extract repo as .zip

    2. Navigate to the Chrome Manage Extensions page

    3. Enable developer mode

    4. Cick 'Load unpacked' and select the folder 'Overwatch'

    5. Make sure the Overwatch extension is enabled and refresh any pre-existing YouTube tabs
